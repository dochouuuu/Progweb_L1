# Full-Stack Image Gallery Application

This project involves the **development of a full-stack web application for displaying and managing image galleries**, built **from scratch (without any frameworks)** during my **first year** as part of a Web Programming course.

The application allows users to browse image galleries, authenticate securely, and interact through a dynamic commenting system.

## Main Features

- Dynamic display of image galleries  
- User authentication system  
- User session management  
- Commenting system associated with images  
- Handling of HTTP requests (GET / POST)  
- Data persistence using a relational database  


## Architecture

The application follows a **full-stack architecture**, including:

- **Frontend**  
  - HTML / CSS / JavaScript  
  - Dynamic content rendering  

- **Backend**  
  - HTTP request handling  
  - Authentication and session management  
  - Server-side business logic  

- **Database**  
  - Storage of users  
  - Image and comment data  
  - Relational links between entities  

## Technologies Used

- HTML  
- CSS  
- JavaScript
- Backend language Node.js
- Relational database PostgreSQL
